
```java

// 文件名: Mytest.java
package org.example;

//首先声明一个接口关键字 接口里面放很多方法 不需要body 相当于定义了一系列方法的规范
interface Test {
    void test1();
}

//可以用一个类去实现这个接口 使用@Override会自动检查是否和接口的方法参数有什么差错


public class Mytest implements Test {

    @Override
    //然后就可以实现具体的方法body
    public void test1() {
        System.out.println("test1");
    }

    // 将 main 方法放到类中
    public static void main(String[] args) {
	    // 这个是多态 也就是说Test是接口 Mytest是具体实现类
        Test t1 = new Mytest();
        t1.test1();
    }
}
```
